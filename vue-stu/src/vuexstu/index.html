学习vuex状态管理

安装

npm install vuex --save

Store

  • 通过Vuex.store创建Store实例 注册到vue实例中
  • 在根实例中注册 store 选项，该 store 实例会注入到根组件下的所有子组件中，且子组件能通过 this.$store 访问到。

Getter
